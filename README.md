# Projet Educore - Gestion de Base de Données

Ce projet a été réalisé dans le cadre de mon Bachelor 3 à l'ÉCOLE-IT. Il simule une plateforme d'apprentissage en ligne.

## Structure du projet
- `1_structure.sql` : Création des tables et contraintes.
- `2_donnees.sql` : Jeu de données de test.
- `3_audit.sql` : Scripts de contrôle qualité.
- `4_analyses.sql` : Requêtes Marketing, Pédagogie et Finance.
- `5_vues.sql` : Vues simplifiées pour le reporting.
- `6_optimisation.sql` : Indexation et analyses EXPLAIN.
- `7_transactions.sql` : Gestion de la sécurité des données (ACID).

## Comment exécuter le projet
1. Exécuter les scripts dans l'ordre numérique dans MySQL Workbench.
